# rescore

In the run.py file input all of the needed variables:

``
files=['Rep1.csv','Rep2.csv','Rep3.csv']
lengths=[100,100,100]
dotfile='./graph.dot'
datatype='continuous'
ncore=20
outfile='out.csv'
``
Then the rescoring will run and output a csv file with all the rescored values
Simply run the python file

``
python run.py
``
